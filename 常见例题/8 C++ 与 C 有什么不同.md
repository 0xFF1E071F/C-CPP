# C++ 与 C 有什么不同

## 语言本身
- C
	是结构化语言
- C++
	是面象对象设计的语言

## 侧重点
- C

	C重点在于算法和数据结构，

	C是C++的子集，

	C首要考虑的问题是如何通过一个过程，对输入进行运算处理，得到输出。

- C++
	重点在于构造合理的对象模型，让这个模型能够配合对应的问题，通过获取对象信息的状态得到输出或实现过程控制。

## 设计层面
- C
C实现了C++中过程化控制及其他相关功能

- C++
	C++相对于原来的C有所加强，引入了重载、内联函数、异常处理等，更是拓展了面向对象设计的内容，如类、继承、虚函数、模板和包容器类等。

	在C++中，不仅需要考虑数据封装，还需要考虑对象粒度的选择、对象接口的设计和继承、组合和继承的使用等问题，

	相对于C，C++包含了更丰富的设计概念。
